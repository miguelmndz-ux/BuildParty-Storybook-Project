✦ Component tier now references human-readable semantic names directly — no more intermediate size-control-* layer.
What changed
Primitives now contain only colors + sizing-N scale
padding, radius, font tokens moved to semantic as named scales

Semantic also includes adjacent tokens (none, xxs, xs, md, full) that future components will reuse
